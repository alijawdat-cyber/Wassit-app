## 🚀 البدء السريع

### **1. 📥 نسخ المشروع**
```bash
git clone https://github.com/alijawdat-cyber/Wassit-app.git
cd Wassit-app
```

### **2. 🛠️ إعداد البيئة**
```bash
# تأكد من تثبيت Flutter 3.19+
flutter --version

# تثبيت dependencies
cd wassit_app
flutter pub get

# تشغيل التطبيق
flutter run -d chrome  # أو أي منصة أخرى
```
